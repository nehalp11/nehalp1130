### 🌟 Featured Projects

#### 🔥 TrendSense — Emerging Indian Fashion Trend Detection
> Python • PyTorch • BERT • CLIP • DBSCAN • Streamlit • Google Trends • Reddit API

Detects emerging Indian fashion micro-trends **6–8 weeks** before mainstream platforms by fusing signals from Google Trends, Reddit, and Pinterest. Designed **Recency Attention Reweighting (RAR)** — a novel modification to BERT's attention formula that separates present-tense discovery posts from historical references, reducing false trend alerts.
